Learn Spanish is a basic tool to help you learn Spanish. As someone who loves learning languages, I created this simple web app as a way to practice my Spanish vocabulary and verb conjugations.

## Usage

To use Learn Spanish, simply visit the [live demo](https://zeckdude.github.io/learnspanish/) and start practicing. The app includes a list of common Spanish vocabulary words, as well as a tool for practicing verb conjugations in various tenses.

## Features

- Practice Spanish vocabulary
- Practice Spanish verb conjugations in various tenses
